## Controls

| Binding | Action |
|:-------:|:-------|
| WASD | Move |
| Space | Jump |
| Mouse Left | Shoot |
| Mouse Right | Aim down sight |
| R | Reload |
| Tab | Open minimap |
| Escape | Exit |


## Dependency
To use the project, you need to install [FMOD for Unity](https://assetstore.unity.com/packages/p/fmod-for-unity-161631)
